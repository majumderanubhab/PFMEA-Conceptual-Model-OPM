# PFMEA OPM Model

This repository contains an Object-Process Methodology (OPM) model of the Process Failure Mode and Effects Analysis (PFMEA) process.

The model captures the complete PFMEA workflow, including:

- Scope Definition
- Structure Analysis
- Function Analysis
- Failure Analysis
- Risk Analysis
- Optimization

It formalizes relationships between process items, process steps, work elements (Human, Machine, Material, Method, Measurement, Environment), failure modes, failure causes, failure effects, and risk attributes (Severity, Occurrence, Detection, Action Priority).
